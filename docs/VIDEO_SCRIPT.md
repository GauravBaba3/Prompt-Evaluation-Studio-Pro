# 3-Minute Demo Video Script

## Prompt Evaluation Studio Pro — Walkthrough

**Total Duration:** ~3 minutes  
**Format:** Screen recording with voiceover

---

### [0:00 - 0:20] Introduction

**Visual:** Title card → Login page

**Script:**  
"Hi, I'm [Your Name]. Today I'll demo Prompt Evaluation Studio Pro — a platform I built for professional prompt engineering using Python, Streamlit, and Google Gemini. Let me show you how it works."

---

### [0:20 - 0:40] Login & Dashboard

**Visual:** Sign up or login → Dashboard loads

**Script:**  
"First, I'll log in to my account. The dashboard gives an instant overview — total prompts, experiments, average response time, and success rate. I can see my prompt categories and recent experiments at a glance."

---

### [0:40 - 1:10] Prompt Playground

**Visual:** Navigate to Playground → Enter prompt → Click Run

**Script:**  
"Let's head to the Playground. Here I configure the system prompt, user prompt, and input variables. I can adjust temperature, top-p, max tokens, and even enable streaming or JSON mode. I'll run this prompt now... and here's the AI response rendered in markdown. Every run auto-saves as an experiment."

---

### [1:10 - 1:40] AI Evaluation

**Visual:** Navigate to Evaluation → Click Evaluate

**Script:**  
"Next, the Evaluation Engine. I paste my prompt and response, click Evaluate, and Gemini scores the output on nine criteria — accuracy, completeness, hallucination risk, grammar, and more — giving an overall score out of 100. It also explains the score and suggests an optimized prompt."

---

### [1:40 - 2:10] Prompt Comparison

**Visual:** Navigate to Comparison → Run 2 variants

**Script:**  
"The Comparison module lets me test two to five prompt variants with the same input. It measures response time, output length, token usage, and quality scores. The winner gets a badge based on weighted scoring."

---

### [2:10 - 2:30] Optimizer & Version Control

**Visual:** Quick flash of Optimizer results → Version history

**Script:**  
"The Optimizer rewrites my prompt into multiple improved versions with explained changes. Version Control tracks every change with snapshots, diffs, and restore capability — like Git for prompts."

---

### [2:30 - 2:50] Analytics & Export

**Visual:** Analytics charts → Export page

**Script:**  
"Analytics shows daily usage trends, category distribution, and response times in interactive Plotly charts. The Export center generates PDF, CSV, Markdown, JSON, and HTML reports."

---

### [2:50 - 3:00] Closing

**Visual:** Dashboard or GitHub README

**Script:**  
"That's Prompt Evaluation Studio Pro — a complete platform for prompt engineering with 12 modules, 23 passing tests, and Docker deployment. Check out the GitHub repo for source code and documentation. Thanks for watching!"
