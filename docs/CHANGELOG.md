# Changelog

All notable changes to Prompt Evaluation Studio Pro.

## [1.0.0] - 2026-06-26

### Added
- Complete Streamlit web application with 12 feature modules
- Local authentication with bcrypt password hashing
- Prompt Playground with streaming, JSON mode, and safety settings
- Prompt Comparison (2-5 variants) with winner scoring
- Version control with diff viewer, tags, favorites, pinning
- Prompt Library with search, filter, templates, sorting
- Experiment Manager with status tracking and export
- AI Evaluation Engine with 9 criteria scoring (0-100)
- Prompt Optimizer with multi-version output
- Analytics Dashboard with Plotly charts
- Export center (PDF, CSV, Markdown, JSON, HTML)
- Admin settings (API config, backup, restore, clear DB)
- Docker and Docker Compose support
- Complete test suite (23 tests)
- Full documentation and portfolio materials

### Technical
- Python 3.12+ with type hints throughout
- SQLAlchemy ORM with SQLite
- Google Gemini via google-genai SDK
- Pydantic v2 schemas
- Modular service-oriented architecture
