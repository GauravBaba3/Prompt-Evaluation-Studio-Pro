# Screenshot Capture Guide

Capture these screenshots after running `streamlit run app.py` and saving them to `assets/screenshots/`.

## Prerequisites

1. Run the application locally
2. Create an account and log in
3. Configure a valid Gemini API key in Admin Settings
4. Load built-in templates from Library
5. Run at least one prompt in Playground to generate data

## Capture Tips

- Use 1920×1080 resolution
- Hide browser bookmarks bar
- Ensure sample data is populated before capture
- Use Windows Snipping Tool or ShareX
- Save as PNG to `assets/screenshots/`
- Name files exactly as listed above for documentation references
